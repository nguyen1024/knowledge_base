## How to Create PDF Files on Windows

* Select images.
* Right click the first image.
* Select print.

## How to Create PDF Files Using IrfanView

* Reduce images size to 25%.
* Install GhostScript.
* Open IrfanView Thumbnails.
* Select images.
* Right click.
* Select "Start Multipage-PDF dialog with selected files... (PlugIn)
* Select "Options".
* Select "Imaged size:" "Optimal (Fit to page)".
* Select "Image position:" "Centered".
* Select "Colur images:" "JPEG - best quality (95)".
* Select "Grayscale images:" "JPEG - best qualuty (95)".
