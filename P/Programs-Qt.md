## Standalone

- Open "Qt 5.13.1 (MinGW 7.3.0 64-bit)" from the Windows Start menu.
- Option 1
  - Generate a make file.
  - Build.
  - Enter "windeployqt alpha.exe"
- Option 2
  - Create a folder somewhere.
  - Enter "windeployqt --dir /path-to-folder /path-to-exe/alpha.exe"
